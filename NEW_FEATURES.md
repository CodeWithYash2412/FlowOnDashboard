# New Features Quick Reference

## 🎹 Keyboard Shortcuts (NEW)

### Edit Operations
- **Ctrl+C** or **Cmd+C** - Copy selected element
- **Ctrl+V** or **Cmd+V** - Paste from clipboard
- **Ctrl+D** or **Cmd+D** - Duplicate selected element
- **Ctrl+A** or **Cmd+A** - Select all elements

### Undo/Redo
- **Ctrl+Z** or **Cmd+Z** - Undo
- **Ctrl+Y** or **Cmd+Shift+Z** (Mac) - Redo
- **Delete** or **Backspace** - Delete selected elements

---

## 🎨 Enhanced Color Picker (NEW)

### Features
- **10 Predefined Colors**: Black, Red, Blue, Green, Yellow, Purple, Pink, Cyan, Orange, Gray
- **Custom Color Picker**: Click the color input at the bottom of the color palette to select any color
- **Live Preview**: Shape preview updates in real-time as you change colors

### How to Use
1. Select the Shapes tool
2. Open the color palette on the left
3. Click any color OR use the custom color picker to select a custom color
4. Draw shapes - they'll use your selected color

---

## ✏️ Text Formatting Controls (NEW)

### Available Controls
- **Font Size**: Adjust 8px - 72px with interactive slider
- **Bold**: Toggle bold text
- **Italic**: Toggle italic text

### How to Use
1. Select the Text tool from the left toolbar
2. The text formatting panel appears on the left
3. Adjust font size using the slider
4. Toggle bold/italic with checkboxes
5. Click on canvas to add text with your settings

---

## 🪟 Placeholder Creator (NEW)

### Quick Placeholder
- **Method 1**: Click the **+** button (plus icon) in bottom-right corner
- **Method 2**: Media menu → Placeholder

### Features
- Creates a labeled placeholder box
- Default size: 100x50 pixels
- Repositionable and resizable
- Great for prototyping layouts

## 🎯 Copy & Paste Workflow (NEW)

### Method 1: Keyboard Shortcuts
1. Select element on canvas
2. Press **Ctrl+C** to copy
3. Press **Ctrl+V** to paste
4. Move pasted element as needed

### Method 2: UI Button
1. Select element
2. Click **Copy** button in bottom-right
3. Manually paste via Ctrl+V

### Method 3: Duplicate (Fastest)
1. Select element
2. Press **Ctrl+D** to instantly duplicate
3. Positioned slightly offset from original

## 💡 Pro Tips

1. **Quick Duplicate**: Use **Ctrl+D** instead of copy-paste for faster workflow
2. **Select All**: **Ctrl+A** is great before applying styles to everything
3. **Shape Size**: Drag the border-width slider to get perfect shapes
4. **Custom Colors**: Use the color picker for brand-specific colors
5. **Text Sizes**: Font size 16-24px works best for readable text

## 🔧 Common Workflows

### Wireframing
1. Add Rectangle shapes for layout boxes
2. Use Text tool for labels
3. Use Arrows to show flow
4. Ctrl+D to duplicate components

### Flowcharts
1. Add shapes (rectangle, diamond, etc.)
2. Use Arrow tool to connect
3. Add text labels
4. Adjust colors for different categories

### Mind Mapping
1. Start with central shape
2. Add branches using arrows
3. Add text for ideas
4. Use colors to group concepts

### Prototyping
1. Create wireframe with placeholders
2. Add text descriptions
3. Draw boxes with arrows for interactions
4. Screenshot and export

## 🐛 Troubleshooting

### Copy/Paste Not Working?
- Ensure you've selected an element first
- Check browser clipboard permissions
- Try the Copy button instead of keyboard

### Colors Not Applying?
- Make sure shape tool is selected
- Check that color panel is open
- Try reselecting the color

### Text Too Small?
- Use the font size slider to increase size (8-72px)
- Default is 16px

### Screenshot Not Capturing?
- Grant browser permission to capture screen
- Close any security warnings
- Try a smaller screen area first
